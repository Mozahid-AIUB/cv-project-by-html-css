# 📄 Personal CV Website — Md. Mozahidul Islam

A professional and responsive personal CV website built using **HTML5** and **CSS3**.
This project represents my digital resume, showcasing my education, technical skills, certifications, and professional profile as a Computer Science & Engineering student preparing for Software Engineering roles.

---

## 🚀 Project Overview

This CV website is designed to provide a clean and structured online resume that can be:

* Viewed directly in web browsers
* Exported as a professional PDF
* Shared with recruiters and employers
* Used as a personal portfolio foundation

The layout focuses on readability, simplicity, and professional presentation.

## 🖨 Export CV as PDF

1. Open `index.html` using **Google Chrome** or **Microsoft Edge**
2. Press **Ctrl + P**
3. Select **Save as PDF**
4. Paper size → **A4**
5. Enable **Background graphics**
6. Save file as:

```
Md_Mozahidul_Islam_CV.pdf
```
